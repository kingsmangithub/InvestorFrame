# Lean Architecture

## Goal
Make InvestorFrame easy to understand, easy to maintain, and useful for one serious investor before it tries to serve everyone else.

## Core loop

Frame -> Idea -> Decision Gate -> Action / No Action -> Review -> Wisdom Update

## Design rules

- Static first
- Rules before dashboards
- Writing before signals
- Reviews before optimization
- Fewer moving parts over more “intelligence”

## Non-goals

- real-time trading
- broad market terminal behavior
- personalized advice automation
- large live connector surface
- decorative product surfaces
