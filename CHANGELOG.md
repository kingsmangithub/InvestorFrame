## [0.1.0] - 2026-03-09

### Added
- Initial project identity under the name **InvestorFrame**
- Event-driven market intelligence positioning
- Core repo structure for:
  - API
  - CLI
  - config
  - dashboard
  - tests
- Initial README and project framing
- Basic product language around:
  - regime detection
  - sector transmission
  - watchlist analysis
  - scenario simulation
- Landing page and product messaging drafts
- Dashboard and Scenario Lab UX direction
- Brand and design system draft
- Documentation direction for:
  - architecture
  - roadmap
  - contribution
  - security

### Defined
- Product scope as a **research-first**, **event-driven**, **regime-aware** system
- Clear distinction between:
  - market interpretation
  - scenario analysis
  - watchlist alignment
  - non-trading use
- Positioning as a market intelligence and decision-support platform, not a trading engine

### Notes
- This is an early public-facing scaffold
- Core product and design foundations are established
- Future releases will focus on:
  - repo hygiene
  - docs quality
  - product polish
  - scenario workflows
  - API consistency
  - explainability and trust layers
